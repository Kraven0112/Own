Local Storage

.getItem("item_name or keyName") used to access item from local storage.

setItem("item_name",value) is used to set the value of item in local storage.

In local storage we can store string form only.

JSON.parse() is a method used to convert strings into key value pair as in an object.

JSON.stringify() is used to convert object key value into string.

localStorage.clear() is used to delete all data from local storage.

localStorage.removeItem("item_name") is used to remove single item from local storage.
