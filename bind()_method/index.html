call(), apply() and bind() in javascript

call() method is used to call the object as an arguments in javscript.

.apply() mathod is also used to invoke(call) an object.

In call() method we can simply pass additional argument using comma(,) like -- ("Groot",21)

call() method is used to directly invoke the function .

In apply() method we can pass additional arguments by using Arraylist type like -- (["1234","123456"])

bind() method is used to make copy of function and we can invoke it later for any operation
