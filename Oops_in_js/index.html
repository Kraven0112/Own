JavaScript is also an object oriented programming.

There are four main pillars of object oriented programming.

Encapsulation -- It is a mechanism of integrating all usable data into single place and use all that data for different purposes. It is alos known as hiding complexity in programming.

Abstraction -- It is knowns as not showing complexity in programming.

Inheritance -- it is mechanism of extending or inheriting properties of another object in another object

Polymorphism -- it is a mechanism of performing different operation using single form of code in programming.
