Localstorage in javascript

Local Storage is used to store data for future use in javascript.

localStorage.getItem("item_name") is used to get access of data from local storage

localStorage.setItem("Item1-name",item_name) is used to create new item in local storage.

JSON.stringify(object_name) is used to covert object into special string in javascript.

JSON.parse(stringed_object) is used to convert stringify object into object.

localStorage.clear() is used to clear all data from localStorage.

localStorage.removeItem("item_name") is used to single individual data from localStorage.
