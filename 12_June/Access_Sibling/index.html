Access Siblings using javascript methods.

.nextElementSiblind is used to access next sibling which is after current element.

.previousElementSibling is rsed to access previous element which is before current element.

.nextSibling is used to access text object of next element.

.previousSibling is used to access text object of previous element.

.previousElementSibling.innerHTML -- is used to access text value of previous element.

.nextElementSibling.innerHTML -- is used to access text value of next element.
