Browser Object Model (BOM)-it is a part of webAPi

window is a object provided browser to interact with javascript

location.href=" " is one of the important in window object

It is usedfor location show of any address in internet

history is also part of window object in javascript.

history.back() to go back in browser.

history.forward() to go ahead in browser.

history.go(-1) takes one step back in browser.

Like that history.go(1) takes one step forward in browser.

innerHeight shows height of window screen in browser.

innerWidth shows width of window screen in browser

outerHeight show total height of window screen.

outerWidth show total width of window screen.

open('url',name,) is used to open new tab in browser.

close() is used to close tab in browser.

resizeBy(height,width)

resizeTo(height,width)

moveTo() and moveBy() are used to move tab in x-axis and y-axis.

scrollBy(x-axis,y-axis) ,scroll() and scrollTo() are used for scroll

document
