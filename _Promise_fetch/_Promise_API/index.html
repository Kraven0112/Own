Promise Api in javascript

Promise.all() => This api settles all resolved api . If one of the peomises is gets rejected then none of the peomises get resolved.

Promise.allSettled() => This api method is also used to settle the api. It settles all api which are resolved or rejected. It show all the promises even all promises are rejected.

Promise.race() => This api method returns single promise even promise is resolved or rejected. It depends on time where promise gets resolved or rejected.

Promise.any() => This api method also returns single promise. It is also known as promise resolve seeker.

If all the peomises are rejected when using Promise.any(), then it will returns aggregate errors.

Aggregate errros means array of errors.
